Integration Demo

Integrate Knock into your site in 4 simple steps

  1. Include the integration library (v1.1.0) 
    <script src="https://s3.amazonaws.com/prod-knock-integration/knock-integration-1.1.0.min.js"></script>
    knock.init('demo'); // Replace demo with your company id
  2. opens all listings in an iframe 
    knock.open(); // no parameter
  3. knock.open("listingid") - opens listing Le36VKmw0RyPzmQd in an iframe 
    knock.open('5eVPYJxvYwVN3N5b'); // 5eVPYJxvYwVN3N5b a listing id belongs to demo
  4. knock.openScheduling() - opens a slimmed down scheduling view which shows a dropdown with all of your floorplans. 
    knock.openScheduling(); // no parameter
